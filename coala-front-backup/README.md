### Commit prefix

- Add: 신규 기능 추가
- Update: 코드 업데이트
- Refactor: 로직 수정 / 코드 리팩토링(기능 변경X, 버그 처리X)
- Fix: 버그 수정
- Style: 디자인 관련 변경사항
- Docs: 문서 추가 및 수정
- Test: 테스트 코드 추가/수정/삭제
- Config: 환경설정 변경
- Chore: 그 외 자잘한 일
